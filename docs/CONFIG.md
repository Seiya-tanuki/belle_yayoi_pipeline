# CONFIG

## Script Properties（必須）
- BELLE_SHEET_ID
- BELLE_DRIVE_FOLDER_ID
- BELLE_GEMINI_API_KEY
- BELLE_GEMINI_MODEL

## Script Properties（任意・既定）
- BELLE_QUEUE_SHEET_NAME（未設定時は BELLE_SHEET_NAME）
- BELLE_SHEET_NAME（belle_appendRow の既定値は OCR_RAW）
- BELLE_OUTPUT_FOLDER_ID（未設定時は BELLE_DRIVE_FOLDER_ID）
- BELLE_IMPORT_LOG_SHEET_NAME（既定 IMPORT_LOG）
- BELLE_SKIP_LOG_SHEET_NAME（既定 EXPORT_SKIP_LOG）
- BELLE_REVIEW_STATE_SHEET_NAME（既定 REVIEW_STATE）
- BELLE_REVIEW_UI_SHEET_NAME（既定 REVIEW_UI）
- BELLE_REVIEW_LOG_SHEET_NAME（既定 REVIEW_LOG）
- BELLE_STRICT_EXPORT（既定 false）
- BELLE_EXPORT_BATCH_MAX_ROWS（既定 5000）
- BELLE_CSV_ENCODING（既定 SHIFT_JIS）
- BELLE_CSV_EOL（既定 CRLF）
- BELLE_INVOICE_SUFFIX_MODE（既定 OFF）
- BELLE_GEMINI_SLEEP_MS（既定 500）
- BELLE_MAX_ITEMS_PER_RUN（既定 1）
- BELLE_RUN_MAX_SECONDS（既定 240）
- BELLE_RUN_MAX_OCR_ITEMS_PER_BATCH（既定 5）
- BELLE_RUN_DO_QUEUE（既定 true）
- BELLE_RUN_DO_OCR（既定 true）
- BELLE_RUN_DO_EXPORT（既定 false）
- BELLE_FALLBACK_DEBIT_TAX_KUBUN_DEFAULT（既定 対象外）
