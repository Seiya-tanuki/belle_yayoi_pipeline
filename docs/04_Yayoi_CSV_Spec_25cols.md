# 04_Yayoi_CSV_Spec_25cols

本書は弥生会計の「弥生取り込み（インポート）形式（弥生会計05以降）」の25項目仕様を、実装用に固定する。

## 1. 重要事項
1. CSVは**ヘッダなし**（データ行のみ）
2. 区切りは半角カンマ
3. 文字コードは Shift_JIS（CP932）を推奨（顧客サンプルと整合）
4. 金額/日付は半角
5. 最後の項目が空欄の場合は末尾のカンマか "" が必要（ただし v0 では 25列目を必ず入れる）

## 2. 列仕様（1行=1仕訳）
列は 1〜25。v0のデフォルト値も併記する。

1. 識別フラグ（必須 / 4桁）
   1) v0：常に "2000"（伝票以外の仕訳データ）
2. 伝票No.（任意）
   1) v0：空欄
3. 決算（任意）
   1) v0：空欄
4. 取引日付（必須）
   1) v0："YYYY/MM/DD"（issue_date）
5. 借方勘定科目（必須）
   1) v0："仮払金"（固定）
6. 借方補助科目（任意）
   1) v0：空欄
7. 借方部門（任意：スタンダード等では不可）
   1) v0：空欄
8. 借方税区分（必須）
   1) v0：docs/03 の決定結果（例：課対仕入込10%区分80%）
9. 借方金額（必須 / 税込金額）
   1) v0：tax bucket の gross_jpy
10. 借方税金額（任意：税抜処理の場合必須）
   1) v0：空欄（常に税込入力）
11. 貸方勘定科目（必須）
   1) v0："現金"（固定）
12. 貸方補助科目（任意）
   1) v0：空欄
13. 貸方部門（任意）
   1) v0：空欄
14. 貸方税区分（必須）
   1) v0："対象外"
15. 貸方金額（必須 / 税込金額）
   1) v0：借方金額と同額（bucket gross）
16. 貸方税金額（任意：税抜処理の場合必須）
   1) v0：空欄
17. 摘要（任意）
   1) v0："{merchant} {doc_type}" を基本に、必要なら追記
18. 番号（任意）
   1) v0：空欄
19. 期日（任意）
   1) v0：空欄
20. タイプ（必須）
   1) v0：0（仕訳データ）
21. 生成元（任意）
   1) v0：空欄
22. 仕訳メモ（任意）
   1) v0：空欄
23. 付箋1（任意）
   1) v0：空欄（=0扱い）
24. 付箋2（任意）
   1) v0：空欄
25. 調整（必須）
   1) v0："NO"（チェックを付けない）

## 3. 出力行数のルール（税率混在）
1. 単一税率：1行
2. 8%+10%混在：税率ごとに1行ずつ（合計2行）
   1) 各行の借方/貸方金額は、それぞれの bucket gross
   2) 行の合計（貸方金額の総和）は receipt_total_jpy に一致する

## 4. 参考（公式）
1. 25項目仕様：support.yayoi-kk.co.jp / page_id=18545
2. 税区分の記述形式：support.yayoi-kk.co.jp / page_id=18111
